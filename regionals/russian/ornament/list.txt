Хохлома — традиционная роспись по дереву.
Характеризуется золотистым фоном и растительным орнаментом.
Включает красные, черные и золотые цвета.
Используется для украшения посуды и мебели.

=START= { ornament: Гжель / gzhel.png / 1700 } ===
Гжель — традиционная керамическая роспись.
Выполняется синим цветом по белому фону.
Включает растительные и геометрические мотивы.
Является символом русского народного искусства.
=END= { ornament: Гжель / gzhel.png / 1700 } ===
